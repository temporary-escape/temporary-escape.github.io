# Class Spawner

**Module:** `require("engine")`

A class that represents a weighted entity

------------
## Constructors

List of constructors that will create an instance of `Spawner`.

### new()

Default constructor that initializes the weighted entity with no entity and the default weight.

### new(entity, weight, count)

Parametrized constructor that initializes the weighted entity

**Parameters:**

* entity: `string` - The name of the entity
* weight: `number` - The numerical weight of the entity
* count: `number` - How many of this entity we should spawn

------------
## Fields

List of fields within the table `Spawner`.
These variables belong to this class and can be accessed only via an instance of this class.

### entity

The string name of the entity (not the Lua package name!)

**Type:** `string`

### weight

The weight of the entity (default value is 1.0)

**Type:** `number`

### count

How many of this entity we should spawn.

**Type:** `number`
