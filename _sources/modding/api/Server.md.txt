# Class Server

**Module:** `require("engine")`

A class that represents server operations

------------
## Member Functions

List of **member** functions within the table `Server`.
These functions need to be called using the colon symbol on some instance of `Server`.

### set_generator(fn)

Sets the function to be called when generating the universe.

**Parameters:**

* `fn: function` - A callback function that accepts a seed as a number

### start_sector(galaxy_id, system_id, sector_id)

Starts a sector. The function won't do anything if the server is already started.

**Parameters:**

* `galaxy_id: string` - The galaxy ID
* `system_id: string` - The system ID
* `sector_id: string` - The sector ID

### move_player_to_sector(server)

Starts a sector. The function won't do anything if the server is already started.

**Parameters:**

* `server: engine.Server` - The other server
