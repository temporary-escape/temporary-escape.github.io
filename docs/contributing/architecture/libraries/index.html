    Libraries #

    List of libraries used in this game.

    • wren - Wren, an embeddable scripting language. Used for scripting purposes and modding.
    • wrenbind17 - A header only library for binding C++17 to Wren language.
    • nuklear - A single-header ANSI C immediate mode cross-platform GUI library.
    • zlib - Needed by libpng.
    • glfw3 - Window and OpenGL context management.
    • glm - OpenGL header only C++ mathematics library.
    • glad - OpenGL extensions loader header only library.
    • fmt - Formatting library providing a fast and safe alternative to C stdio and C++ iostreams.
    • libxml2 - XML C parser and toolkit developed for the Gnome project.
    • nanovg - NanoVG is small antialiased vector graphics rendering library for OpenGL. Used for rendering simple shapes, text, and the GUI.
    • libpng - Official PNG reference library. Used for loading png assets.
    • cgltf - Single-file/stb-style C glTF loader and writer. Used for loading gltf assets.
    • msgpack-c - MessagePack is an efficient binary serialization format. Used for data structure serialization and deserialization in the networking stack and the database.
    • asio - A cross-platform C++ library for network and low-level I/O programming. Used for low level networking and background workers.
    • stb - Single-file public domain libraries. Only the stb_rect_pack.h part is used for packing images into image atlases.
    • catch2 - A modern, C++-native, test framework for unit-tests. Used for unit tests and functional tests.
    • spdlog - Fast C++ logging library.
    • cli11 - Command line parser for C++11 and beyond that provides a rich feature set with a simple and intuitive interface.
    • freetype - True type font loader and glyph renderer. Used by nanovg to load and render fonts.
    • rocksdb - An embeddable persistent key-value store for fast storage. Used as a data store (with msgpack) to persistently store world data.
    • openssl - Toolkit for general-purpose cryptography and secure communication. Used on top of a custom networking stack with asio.
    • libuuid (Linux & Mac OSX only) - Used for generating UUID strings.
    • liblzma - Free general-purpose data compression software with high compression ratio.
