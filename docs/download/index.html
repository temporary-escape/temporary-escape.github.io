    System Requirements #

    These are the minimum system requirements. You must have Vulkan 1.1 (or higher) compatible graphics card and a 64-bit operating system.

    • OS: Windows 10 64-bit, Ubuntu/Debian/Manjaro 64-bit, or MacOS
    • Display: 1680x1050
    • Processor: Intel Core i5 or AMD Ryzen 3
    • Memory: 4 GB RAM
    • Graphics: Intel HD Graphics 500, NVidia GTX 650 Ti, or AMD Radeon R7
    • Vulkan: 1.1 minimum
    • Storage: 500 MB available space
