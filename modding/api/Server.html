Class Server#

Module: require("engine")

A class that represents server operations

Member Functions#

List of member functions within the table Server. These functions need to be called using the colon symbol on some instance of Server.

move_player_to_sector(player_id)#

Moves a player to sector and starts the sector if it is not running

Parameters:

  • player_id: The - ID of the player to move

add_sector_type(name, type)#

Adds a new sector type, or overwrites an existing sector type, for the galaxy generator.

Parameters:

  • name: Name - of the sector type

  • type: An - instance of SectorType
